# LUE

LUE is software which can be used to develop models that simulate geographical systems, both small and very
large. Key aspects that, in combination, sets LUE apart from alternatives:

- **Scalable** During model development, small datasets can be used, on a laptop, for example. For production
runs, the model can be copied, unchanged, to the production hardware, like a compute cluster, and run there.
LUE uses parallel computing, distributed computing, and parallel I/O for this.
- **Easy to use** High level API. No need to understand how high performance computing works. Developing
models using LUE is as easy as using Python's NumPy package.
- **Multiple language bindings** Use your favorite programming language. Currently supported languages are C++
and Python.
- **Portable** LUE works on all relevant platforms.
- **Free to use**
- **Open source** The source code is available under liberal licensing terms (MIT).
- **Active** The LUE project is in active development. If you want to participate (welcome!), then more
information can be found on the [LUE website](https://lue.computationalgeography.org).
