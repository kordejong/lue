## Terminology

Logo
: Graphic that identifies a **brand**. Can take many forms, like a wordmark, symbol, icon.

Icon
: Not a logo. Typically flat, minimalistic, stylised.

Brandmark / symbol
: Logo without text. Not all symbol logos are icons but they *can* be used as an icon.

Wordmark / logo type
: Text logo

Lettermark
: Logo made with 1-2 letters

Brandmark
: Logo combining graphic + wordmark
